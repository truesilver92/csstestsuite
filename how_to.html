CSS How To...

There are three ways of inserting a style sheet:

  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet

Looks like this
[link rel="stylesheet" type="text/css" href="mystyle.css" /]
(It goes inside the head of the html page)
More than one may be referenced

Internal Style Sheet

Goes inside the style element within the head of the html document

Inline Styles

Goes inside the individual elements it effects. Do it using the style attribute.

When using more than one style sheet the most specific property will be recognized
